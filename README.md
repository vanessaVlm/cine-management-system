# Sistema de Gerenciamento e Reserva de Cinema (CINE)

## 🎓 Contexto Acadêmico
* **Disciplina:** Programação Orientada a Objetos
* **Semestre:** 2º Semestre
* **Objetivo:** Projeto prático desenvolvido para aplicar os pilares da Orientação a Objetos (POO) e construção de interfaces gráficas (GUI).

## 🚀 Sobre o Projeto
Aplicação desktop desenvolvida em Java utilizando a biblioteca Swing. O sistema automatiza o fluxo operacional de um cinema, permitindo o controle de perfis de acesso, cadastro de usuários e gerenciamento de catálogo de filmes.

## ✨ Funcionalidades
* **Autenticação:** Telas de Login e Cadastro de usuários.
* **Controle de Acesso (RBAC):** Fluxos de navegação personalizados para Administradores e Clientes.
* **Painel Administrativo:** Gestão completa do catálogo de filmes, com suporte para exibição dinâmica de cartazes.
* **UX/UI:** Interface intuitiva com alternância de visibilidade de senhas e feedback visual nos botões.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java SE
* **Interface Gráfica:** Java Swing / AWT
* **Paradigma:** Programação Orientada a Objetos
